Find Me
STAGE 1 : 10 images will be shown, followed by a tone. The goal of the game is to guess these 10 pictures and the tone at the final stage of the game. But in order to go to the final stage user should pass the stages coming before that. According to the scores the user gets in these sub stages , user will be provided with some clues to find the correct images in the final Stage.

STAGE 2 : In this stage user has to break a 4 digit code within 10 tries. where each digit is different. if the user fails this stage he/she will lose the whole game. Red Square indicates that one of the guessed digit is correct, and it is in a correct position. Grey Square indicates that one of the guessed digit is correct, and it is in a Wrong position. However The order of the Red and Grey squares does not indicates the exact position of the guessed digit.

STAGE 3 : In this stage , user will be given 10 tones similar to the tone that was played on the first stage. user has to find the correct one. user will lose the whole game if user fails to guess the tone.
